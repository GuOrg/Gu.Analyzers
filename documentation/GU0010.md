## Description

Assigning same value does not make sense and is sign of a bug.

## Motivation

While not a common bug this happens:

```C#
public class Foo
{
    public Foo(int a)
    {
        this.A = A;
    }

    public int A { get; private set; }
}
```

## How to fix violations

You probably meant:

```C#
public class Foo
{
    public Foo(int a)
    {
        this.A = a;
    }

    public int A { get; private set; }
}
```
